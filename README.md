# Autonomous Business Analyst Agent

## What I Built
I built an autonomous AI agent that acts as a business analyst. It takes any CSV business data, cleans it, analyzes it, generates insights using AI, and produces a full business report automatically.

## What It Does
- Accepts any CSV file upload
- Cleans the data automatically (duplicates, missing values)
- Analyzes trends, correlations, and top categories
- Generates AI-powered business insights using LLaMA 3.3 70B
- Displays charts and visualizations
- Produces a written business report

## Tools Used
- Python & Pandas - data cleaning and analysis
- Groq API (LLaMA 3.3 70B) - AI insights and report generation
- Streamlit - interactive web app

## What I Learned
This project taught me how to build a multi-agent AI pipeline where each agent has a specific role — profiling, analyzing, generating insights, and writing reports — all connected in one autonomous flow.
